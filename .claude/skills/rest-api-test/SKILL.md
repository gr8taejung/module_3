# REST API TEST 사용법
작업한 API에 대한 테스트 코드를 작성합니다.

## TEST 순서
- REST API 리스트 업하고, 필요한 인자와 타입을 정리합니다
- 각 API 마다 테스트용 인자를 대입하면서 호출하여 테스트합니다
  - 필요 조건을 모두 제공하여 정상 동작 여부를 확인
  - 필수 인자, 선택 인자 제공 여부에 따른 동작 확인
- 테스트 결과는 `.claude/docs/test.md`에 작성합니다.
  - [] 의 체크박스를 사용하여, 직관적으로 표현
